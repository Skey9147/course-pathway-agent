# Course Pathway AI Agent

## Overview
This project is an AI-powered Course Pathway Assistant built using Langflow and IBM Granite. It helps students generate personalized learning roadmaps, resources, and career guidance.

## Features
- Student profile analysis
- 6-month learning roadmap generation
- Resource recommendations
- Career guidance and project suggestions

## Tech Stack
- IBM Granite LLM
- Langflow
- Python
- Prompt Engineering

## How it works
Student input → Multi-Agent Flow → Learning Path → Resources → Career Guidance

## Output
Generates structured personalized roadmap for students.
